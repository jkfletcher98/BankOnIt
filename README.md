# BankOnIt
ATM-esque app

## HasMenu
```
string menu()
	menu gets 4 options based on what menu is currently open
	string response gets input from menu
	return response

void start()
	Asks for login info
	if customer login
		login successful
		go to customer menu
	if admin login
		login successful
		go to admin menu
	else
		login unsuccessful
```
